# 🏥 DocAppoint — Doctor Appointment Manager

A doctor appointment booking platform where users can browse doctors, book appointments, and manage their healthcare online.

## Features

- Email/Password and Google authentication with Firebase
- Browse and search doctors by name and specialty
- Book, update, and cancel appointments easily
- Personal dashboard to manage bookings and profile
- JWT-based security for all protected routes

## Tech Stack

Next.js, Tailwind CSS, Firebase, Axios, Swiper.js
